# Branch locator app

This app was implement with the help of the google maps api.
## please include the following to run correctly locally 
1. **.env** file
2. In the **.env** file, **VITE_GOOGLE_MAPS_API_KEY** with a google api key and **VITE_MAP_ID**
with the ID of the Map

## How to run
Run the following commands
1. npm i
2. npm run dev
3. access on localhost:5173
